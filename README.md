# 📝 Online Exam System

A clean, interactive browser-based exam application built with vanilla JavaScript, HTML, and CSS. Features a countdown timer, question navigation, answer marking, and a dynamic sidebar.

---

## 🚀 Features

- ⏱️ Countdown timer with danger warning at 2 minutes
- 🔀 Questions shuffled randomly on every load
- ✅ Answer selection with visual feedback
- ⚑ Mark/Unmark questions for review
- 🧭 Sidebar with live question status (current, answered, marked)
- 📦 Questions loaded from external `questions.json` file
- 📱 Responsive layout

---

## 📁 Project Structure

```
exam-project/
│
├── home.html          # Main HTML file
├── style.css           # Stylesheet
├── exam.js             # Exam Page JavaScript logic
└── questions.json      # Question bank (fetched at runtime)
└── grades.html         # Grade Page JavaScript logic
└── timeout.html        # Timeout Page JavaScript logic
```
